Apresentação do site

Este site reúne os trabalhos da disciplina de Interface Web. A cada nova atividade, uma nova página será adicionada para documentar o processo de desenvolvimento, as decisões de design e o resultado final de cada entrega.

Objetivo

Organizar os projetos em um espaço único, com navegação clara e layout consistente, para facilitar a visualização da evolução dos trabalhos durante o semestre.

Estrutura do conteúdo

O menu superior contém atalhos para todas as páginas de trabalhos. Cada trabalho é desenvolvido com foco em boas práticas de HTML, CSS e JavaScript.

Recursos utilizados neste trabalho
